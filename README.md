# YouTube Medya İndirici 

YouTube videolarını indirmek ve yüksek kaliteli MP3 ses dosyalarını ayıklamak için geliştirilmiş, son kullanıcının bilgisayarına hiçbir kurulum yapmasını gerektirmeyen, modern bir Windows masaüstü uygulamasıdır.

##  Özellikler
- **Kurulumsuz EXE:** Son kullanıcının bilgisayarında Python veya FFmpeg kurulu olmasına gerek yoktur. Her şey tek dosya içindedir.
- **Format Seçimi:** İster yüksek kaliteli MP3 (Ses), ister MP4 (Video) olarak indirin.
- **Dinamik Kalite Desteği:** 1080p, 720p, 480p ve 360p gibi farklı çözünürlük seçeneklerini destekler.
- **OneDrive Uyumlu:** İndirilen dosyaları, bilgisayarda OneDrive aktif olsa bile doğrudan gerçek Windows Masaüstü'ne kaydeder.

##  Kullanılan Teknolojiler (Tech Stack)
- **Görsel Arayüz (GUI):** PyQt6
- **İndirme Motoru:** yt-dlp
- **Ses/Video İşleme (FFmpeg):** imageio-ffmpeg
- **Derleyici:** PyInstaller

##  Nasıl İndirilir ve Çalıştırılır?
1. Bu GitHub deposunun sağ tarafında bulunan **Releases (Sürümler)** bölümüne gidin.
2. `media_indirici.exe` dosyasını bilgisayarınıza indirin.
3. Dosyaya çift tıklayın, linki yapıştırın ve indirmeyi başlatın!
